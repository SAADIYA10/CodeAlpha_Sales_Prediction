# CodeAlpha_Sales_Prediction
This project was developed as part of the CodeAlpha Data Science Internship.  The project demonstrates the complete data science workflow, including data preprocessing, exploratory data analysis, model training, prediction, and performance evaluation.

The Objective is to Predict product sales using advertising data, to Analyze the relationship between advertising expenditure and sales, Build and evaluate a machine learning regression model, Understand the complete machine learning workflow.

The dataset contains advertising expenditure across different marketing channels like TV Advertising, Radio Advertising, Newspaper Advertising.THe Target Variable Sales.The Technologies Used Python, Pandas, NumPy, Matplotlib, Scikit-learn, Jupyter Notebook

Project Workflow:
1. Import Libraries
2. Load Dataset
3. Data Exploration
4. Data Cleaning and Preprocessing
5. Data Visualization
6. Split Dataset into Training and Testing Sets
7. Train the Regression Model
8. Evaluate Model Performance
9. Predict Sales

The Machine Learning Model are:A regression model was trained to predict sales based on advertising expenditure. The model was evaluated using standard regression metrics to measure prediction performance.

The Features are  Data preprocessing, Exploratory Data Analysis (EDA), Regression model training, Sales prediction, Model evaluation, Easy-to-understand workflow.

Final Results:The regression model successfully predicted sales based on advertising expenditure, demonstrating how machine learning can be used for business forecasting and decision-making.

Future Improvements Compare multiple regression algorithms, Perform feature engineering, Improve model accuracy through hyperparameter tuning, Deploy the model as a web application.

The Conclusion of this project demonstrates the practical application of machine learning in predicting sales using advertising data. It provides hands-on experience with regression analysis, data preprocessing, model training, and evaluation.
